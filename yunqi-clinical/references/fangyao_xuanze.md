## 一、五运太过不及的参考方药

### 1.1 木运太过

- **治法**：平肝息风
- **参考方药**：天麻钩藤饮、镇肝熄风汤类
- **方义**：
  - 天麻钩藤饮：平肝息风，清热活血
  - 镇肝熄风汤：镇肝息风，滋阴潜阳
- **适用病证**：肝气偏盛所致眩晕、头痛、胁痛、震颤等
- **兼顾扶脾**：可配四君子汤类健脾益气

### 1.2 木运不及

- **治法**：养肝柔筋
- **参考方药**：补肝汤、四物汤类
- **方义**：
  - 补肝汤：养血滋阴，柔肝舒筋
  - 四物汤：补血调血，养肝体
- **适用病证**：肝气不足所致视力减退、肢体麻痹、拘急等
- **兼顾抑肺**：可配泻白散类清肺泻金

### 1.3 火运太过

- **治法**：清心泻火
- **参考方药**：导赤散、泻心汤类
- **方义**：
  - 导赤散：清心利水，导热下行
  - 泻心汤：苦寒泻火，清心除烦
- **适用病证**：心火偏盛所致心烦、口疮、咳血等
- **兼顾扶肺**：可配沙参麦冬汤类润肺益气

### 1.4 火运不及

- **治法**：温养心阳
- **参考方药**：保元汤、桂枝甘草汤类
- **方义**：
  - 保元汤：益气温阳，补养心脾
  - 桂枝甘草汤：温通心阳，平冲降逆
- **适用病证**：心气不足所致心悸、畏寒、胸闷等
- **兼顾抑肾**：可配真武汤类温肾化水

### 1.5 土运太过

- **治法**：健脾化湿
- **参考方药**：平胃散、胃苓汤类
- **方义**：
  - 平胃散：燥湿运脾，行气和胃
  - 胃苓汤：健脾利水，祛湿和胃
- **适用病证**：脾湿偏盛所致腹胀、水肿、小便不利等
- **兼顾温肾**：可配金匮肾气丸类温肾化水

### 1.6 土运不及

- **治法**：健脾益气
- **参考方药**：四君子汤、参苓白术散类
- **方义**：
  - 四君子汤：益气健脾，培补中州
  - 参苓白术散：健脾益气，渗湿止泻
- **适用病证**：脾气不足所致食少、倦怠、泄泻等
- **兼顾平肝**：可配逍遥散类疏肝健脾

### 1.7 金运太过

- **治法**：润肺清燥
- **参考方药**：清燥救肺汤、沙参麦冬汤类
- **方义**：
  - 清燥救肺汤：清燥润肺，益气养阴
  - 沙参麦冬汤：清养肺胃，生津润燥
- **适用病证**：肺燥偏盛所致咳喘、皮肤干、便秘等
- **兼顾养肝**：可配四物汤类养肝柔筋

### 1.8 金运不及

- **治法**：补肺益气
- **参考方药**：补肺汤、玉屏风散类
- **方义**：
  - 补肺汤：补肺益气，固表养阴
  - 玉屏风散：益气固表，止汗防感
- **适用病证**：肺气不足所致气短、自汗、易感冒等
- **兼顾清心**：可配导赤散类清心泻火

### 1.9 水运太过

- **治法**：温阳化水
- **参考方药**：真武汤、苓桂术甘汤类
- **方义**：
  - 真武汤：温阳利水，健脾化湿
  - 苓桂术甘汤：温化痰饮，健脾利湿
- **适用病证**：肾寒偏盛所致寒战、水肿、心悸等
- **兼顾通心**：可配桂枝甘草汤类温通心阳

### 1.10 水运不及

- **治法**：温补肾阳
- **参考方药**：金匮肾气丸、右归丸类
- **方义**：
  - 金匮肾气丸：温补肾阳，化气行水
  - 右归丸：温补肾阳，填精止遗
- **适用病证**：肾气不足所致腰膝酸软、小便频、畏寒等
- **兼顾健脾**：可配四君子汤类健脾化湿

## 二、六气司天的参考方药

### 2.1 厥阴风木司天

- **治法**：平肝息风，健脾和胃
- **参考方药**：天麻钩藤饮（平肝）合四君子汤（健脾）
- **适用**：上半年风证多发，肝脾不调

### 2.2 少阴君火司天

- **治法**：清心泻火，润肺降逆
- **参考方药**：导赤散（清心）合沙参麦冬汤（润肺）
- **适用**：上半年热证多发，心肺受邪

### 2.3 太阴湿土司天

- **治法**：健脾化湿，温肾利水
- **参考方药**：平胃散（健脾）合金匮肾气丸（温肾）
- **适用**：上半年湿证多发，脾肾不调

### 2.4 少阳相火司天

- **治法**：清火泻热，润肺降逆
- **参考方药**：黄连解毒汤（清火）合沙参麦冬汤（润肺）
- **适用**：上半年火证多发，上焦热病

### 2.5 阳明燥金司天

- **治法**：润肺清燥，养肝柔筋
- **参考方药**：清燥救肺汤（润肺）合四物汤（养肝）
- **适用**：上半年燥证多发，肝肺不调

### 2.6 太阳寒水司天

- **治法**：温肾散寒，通心温阳
- **参考方药**：真武汤（温肾）合桂枝甘草汤（通心）
- **适用**：上半年寒证多发，心肾不交

## 四、运气方剂的选用原则

### 4.1 非固定方原则

- 运气方剂不是固定方，而是根据运气格局提供参考方向
- 须结合患者实际证候辨证选方
- 运气方剂提供的是"方向性参考"，而非"处方模板"

### 4.2 选用步骤

1. 根据大运太过不及确定脏腑方向
2. 根据六气司天在泉确定经络方向
3. 根据治则治法（参 `zhize_zhifa.md`）确定治疗大法
4. 在治疗大法指导下选择参考方药
5. 结合患者实际证候调整选方

### 4.3 配伍原则

- 太过之年：泻本脏 + 补受邪脏 + 兼顾来复
- 不及之年：补本脏 + 泻来乘脏 + 兼顾反侮
- 司天在泉：治所胜之腑 + 兼治所主之腑
- 运气同化年：偏盛更重，用药须更精准
