# 客主加临

## 客主加临的概念

客主加临是将逐年变化的**客气**叠加在固定的**主气**之上，分析二者之间的五行生克关系，以判断各步气候的顺逆变化。

- **主气**：固定不变，反映正常气候规律（详见 [主气与客气](zhuqi_keqi.md)）
- **客气**：逐年变化，反映异常气候变化（由司天在泉决定）

客气叠加于主气之上，如同客至主家，故名"客主加临"。

## 顺逆判断规则

客主加临的顺逆根据客气与主气之间的五行生克关系判断：

| 关系 | 判断 | 含义 |
|------|------|------|
| 客气同主气（同五行） | 相得（顺） | 客主同气，气候平和 |
| 客气生主气 | 相得（顺） | 客助主，气候顺应 |
| 客气克主气 | 不相得（逆） | 客凌主，气候异常 |
| 主气生客气 | 不相得（逆） | 主生客，下生上，气逆 |
| 主气克客气 | 不相得（逆） | 主克客，主客相争 |

### 判断要点

- **相得（顺）**：客气与主气五行相同或客气生主气，气候相对平和
- **不相得（逆）**：客气克主气、主气生客气或主气克客气，气候异常变化

## 相得与不相得年份的意义

### 相得年份

- 六步中多数为相得（顺）
- 气候相对平和，与正常季节规律偏差较小
- 疾病发生相对较少，病情相对较轻

### 不相得年份

- 六步中多数为不相得（逆）
- 气候异常变化大，易出现反季节气候
- 疾病发生较多，病情相对较重

## 五行生克关系速查

```
相生: 木→火→土→金→水→木
相克: 木→土→水→火→金→木

六气→五行:
  厥阴风木→木, 少阴君火→火, 少阳相火→火
  太阴湿土→土, 阳明燥金→金, 太阳寒水→水
```

## 完整示例：2024甲辰年

### 基本推算

- 天干：甲 → 大运 = 土运太过
- 地支：辰 → 司天 = 太阳寒水（三阳）
- 在泉 = 太阴湿土（三阴）

### 主气六步（固定）

| 步次 | 主气 | 五行 |
|------|------|------|
| 初气 | 厥阴风木 | 木 |
| 二气 | 少阴君火 | 火 |
| 三气 | 少阳相火 | 火 |
| 四气 | 太阴湿土 | 土 |
| 五气 | 阳明燥金 | 金 |
| 终气 | 太阳寒水 | 水 |

### 客气六步

太阳寒水司天（索引=5），用推移公式 `KEQI_CYCLE[(5 + step - 3) % 6]`：

| 步次 | 客气 | 五行 | 推算 |
|------|------|------|------|
| 初气 | 少阳相火 | 火 | KEQI_CYCLE[(5+1-3)%6]=KEQI_CYCLE[3] |
| 二气 | 阳明燥金 | 金 | KEQI_CYCLE[(5+2-3)%6]=KEQI_CYCLE[4] |
| 三气 | 太阳寒水 | 水 | KEQI_CYCLE[(5+3-3)%6]=KEQI_CYCLE[5]（司天） |
| 四气 | 厥阴风木 | 木 | KEQI_CYCLE[(5+4-3)%6]=KEQI_CYCLE[0] |
| 五气 | 少阴君火 | 火 | KEQI_CYCLE[(5+5-3)%6]=KEQI_CYCLE[1] |
| 终气 | 太阴湿土 | 土 | KEQI_CYCLE[(5+6-3)%6]=KEQI_CYCLE[2]（在泉） |

### 六步客主加临分析

| 步次 | 主气（五行） | 客气（五行） | 生克关系 | 判断 |
|------|-------------|-------------|----------|------|
| 初气 | 厥阴风木（木） | 少阳相火（火） | 木生火，主生客 | 不相得（逆） |
| 二气 | 少阴君火（火） | 阳明燥金（金） | 火克金，主克客 | 不相得（逆） |
| 三气 | 少阳相火（火） | 太阳寒水（水） | 水克火，客克主 | 不相得（逆） |
| 四气 | 太阴湿土（土） | 厥阴风木（木） | 木克土，客克主 | 不相得（逆） |
| 五气 | 阳明燥金（金） | 少阴君火（火） | 火克金，客克主 | 不相得（逆） |
| 终气 | 太阳寒水（水） | 太阴湿土（土） | 土克水，客克主 | 不相得（逆） |

### 分析结论

2024甲辰年六步客主加临**全部不相得（逆）**，全年气候异常，变化剧烈。太阳寒水司天，上半年偏寒；太阴湿土在泉，下半年偏湿。加之土运太过，湿气偏盛，全年气候以寒湿为主，但各步又有不同程度的逆乱变化。

## 参考公式

```
# 五行相克
SHENG = {"木":"火", "火":"土", "土":"金", "金":"水", "水":"木"}  # A生B
KE = {"木":"土", "土":"水", "水":"火", "火":"金", "金":"木"}    # A克B

# 六气→五行映射
LIUQI_WUXING = {
    "厥阴风木": "木", "少阴君火": "火", "少阳相火": "火",
    "太阴湿土": "土", "阳明燥金": "金", "太阳寒水": "水",
}

# 顺逆判断
def judge_shun_ni(zhuqi_wuxing, keqi_wuxing):
    if zhuqi_wuxing == keqi_wuxing:
        return "相得（顺）"  # 同气
    elif SHENG[keqi_wuxing] == zhuqi_wuxing:
        return "相得（顺）"  # 客生主
    elif KE[keqi_wuxing] == zhuqi_wuxing:
        return "不相得（逆）"  # 客克主
    elif SHENG[zhuqi_wuxing] == keqi_wuxing:
        return "不相得（逆）"  # 主生客
    elif KE[zhuqi_wuxing] == keqi_wuxing:
        return "不相得（逆）"  # 主克客
```
