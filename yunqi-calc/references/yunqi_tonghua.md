# 运气同化（天符/岁会/太乙天符）

## 概念

运气同化是指大运（天干所化之运）与司天之气、岁支（地支五行）之间出现五行属性相同的情况，称为"同化"。同化年气候偏盛，对应脏腑易病。

运气同化分为三种：**天符**、**岁会**、**太乙天符**。

## 天符

**天符**：大运五行与司天五行相同。

判断条件：`大运五行 == 司天五行`

### 示例

己丑年：
- 大运 = 土运（甲己化土，己为土运）
- 司天 = 太阴湿土（丑未→太阴湿土，土）
- 土 = 土 → **天符**

| 己未 | 土 | 太阴湿土（土） | 土 |

## 岁会

**岁会**：大运五行与岁支（地支五行）相同。

判断条件：`大运五行 == 地支五行`

### 地支五行表

| 地支 | 五行 | 地支 | 五行 |
|------|------|------|------|
| 子 | 水 | 午 | 火 |
| 丑 | 土 | 未 | 土 |
| 寅 | 木 | 申 | 金 |
| 卯 | 木 | 酉 | 金 |
| 辰 | 土 | 戌 | 土 |
| 巳 | 火 | 亥 | 水 |

### 示例

甲辰年：
- 大运 = 土运（甲己化土）
- 地支 = 辰（土）
- 土 = 土 → **岁会**

### 岁会年列表（60甲子中共11年）

| 年干支 | 大运 | 岁支 | 岁支五行 | 同气五行 |
|--------|------|------|----------|----------|
| 甲辰 | 土 | 辰 | 土 | 土 |
| 甲戌 | 土 | 戌 | 土 | 土 |
| 丙子 | 水 | 子 | 水 | 水 |
| 丁卯 | 木 | 卯 | 木 | 木 |
| 戊午 | 火 | 午 | 火 | 火 |
| 己丑 | 土 | 丑 | 土 | 土 |
| 己未 | 土 | 未 | 土 | 土 |
| 庚申 | 金 | 申 | 金 | 金 |
| 辛亥 | 水 | 亥 | 水 | 水 |
| 壬寅 | 木 | 寅 | 木 | 木 |
| 癸巳 | 火 | 巳 | 火 | 火 |

注：60甲子中部分干支组合不存在（如庚酉、丁寅、戊巳等），上表已仅列出实际存在的岁会年。其中传统正位岁会年（岁支为该五行正位）有8年：甲辰、甲戌、丙子、丁卯、戊午、己丑、己未、乙酉（乙酉既为天符又为岁会，列入太乙天符）。

## 太乙天符

**太乙天符**：既为天符又为岁会，即大运、司天、岁支三者五行同气。

判断条件：`大运五行 == 司天五行 == 岁支五行`（三天符）

又称**三天符**，指大运、司天、岁支三者同气。

### 示例

己丑年：
- 大运 = 土运（甲己化土）
- 司天 = 太阴湿土（丑→太阴湿土，土）
- 地支 = 丑（土）
- 土 = 土 = 土 → **太乙天符**

### 太乙天符年列表（60甲子中共4年）

| 年干支 | 大运五行 | 司天五行 | 岁支五行 | 是否太乙天符 |
|--------|----------|----------|----------|--------------|
| 己丑 | 土 | 太阴湿土（土） | 丑（土） | 是 |
| 己未 | 土 | 太阴湿土（土） | 未（土） | 是 |
| 乙酉 | 金 | 阳明燥金（金） | 酉（金） | 是 |
| 戊午 | 火 | 少阴君火（火） | 午（火） | 是 |

其余天符年（丙辰、丙戌、丁巳、丁亥、戊子、戊寅、戊申）因岁支五行与大运五行不同，故仅为天符而非太乙天符。

## 运气同化的意义

| 同化类型 | 同气要素 | 气候特点 | 病理倾向 |
|----------|----------|----------|----------|
| 天符 | 大运+司天 | 该年气候偏盛 | 司天所主脏腑易病 |
| 岁会 | 大运+岁支 | 该年气候偏盛 | 岁支所应脏腑易病 |
| 太乙天符 | 大运+司天+岁支 | 气候极偏，该气独盛 | 对应脏腑极易病，病情较重 |

### 各五行同化的病理对应

| 同气五行 | 偏盛之气 | 易病脏腑 |
|----------|----------|----------|
| 木 | 风气 | 肝、脾（木克土） |
| 火 | 热气/火气 | 心、肺（火克金） |
| 土 | 湿气 | 脾、肾（土克水） |
| 金 | 燥气 | 肺、肝（金克木） |
| 水 | 寒气 | 肾、心（水克火） |

## 判断流程

```
1. 计算大运五行（天干化运）
2. 计算司天五行（地支化气→六气→五行）
3. 计算岁支五行（地支本气五行）
4. 判断:
   - 大运 == 司天 → 天符
   - 大运 == 岁支 → 岁会
   - 大运 == 司天 == 岁支 → 太乙天符（三天符）
```
