# 主运与客运

## 主运

主运是每年固定不变的五步运，反映一年五个时段的正常气候规律。主运按五行相生顺序排列：木→火→土→金→水。

### 主运五步

| 步次 | 五行 | 起始节气 | 气候特征 |
|------|------|----------|----------|
| 初运 | 木 | 大寒 | 风气渐生，春温之始 |
| 二运 | 火 | 春分 | 热气渐盛，夏暑将至 |
| 三运 | 土 | 谷雨 | 湿气偏盛，长夏之象 |
| 四运 | 金 | 处暑 | 燥气渐生，秋凉之始 |
| 终运 | 水 | 立冬 | 寒气偏盛，冬寒之象 |

主运五步年年相同，按木火土金水五行相生排列，反映一年五季的正常气候变化。

## 主运太少交替规律

主运的每一步有"太"（太过）与"少"（不及）之分，太少交替排列。太少交替的起点由大运决定：

- **大运太过之年**：大运所在之步为"太"，其余按太少交替推算
- **大运不及之年**：大运所在之步为"少"，其余按太少交替推算

太少交替规则：太→少→太→少→太（或少→太→少→太→少），相邻两步太少相反。

### 太少交替推算方法

1. 确定大运五行及其太过/不及
2. 大运五行对应主运哪一步（初运木、二运火、三运土、四运金、终运水）
3. 该步为太（太过年）或少（不及年）
4. 其余各步按太少交替排列

## 客运

客运是逐年变化的五步运，以大运为初运，按五行相生顺序排列五步。客运反映一年五个时段的异常气候变化。

### 客运五步排列

1. 初运 = 大运五行
2. 二运 = 初运所生之五行
3. 三运 = 二运所生之五行
4. 四运 = 三运所生之五行
5. 终运 = 四运所生之五行

### 客运太少

- **太过年**：初运为太，其余太少交替
- **不及年**：初运为少，其余太少交替

## 示例推算：2024甲辰年（土运太过）

### 大运

甲己化土，甲为阳干 → 土运太过

### 主运五步

大运为土，对应三运（土）。土运太过，故三运为太，太少交替推算：

| 步次 | 五行 | 太少 | 推算依据 |
|------|------|------|----------|
| 初运 | 木 | 太 | 与三运（太）隔两步，太→少→太 |
| 二运 | 火 | 少 | 与初运（太）交替 |
| 三运 | 土 | 太 | 大运所在步，太过→太 |
| 四运 | 金 | 少 | 与三运（太）交替 |
| 终运 | 水 | 太 | 与四运（少）交替 |

主运：**太木 → 少火 → 太土 → 少金 → 太水**

### 客运五步

大运为土，初运=土，按五行相生排列：

| 步次 | 五行 | 太少 | 推算依据 |
|------|------|------|----------|
| 初运 | 土 | 太 | 大运为初运，太过→太 |
| 二运 | 金 | 少 | 土生金，太少交替 |
| 三运 | 水 | 太 | 金生水，太少交替 |
| 四运 | 木 | 少 | 水生木，太少交替 |
| 终运 | 火 | 太 | 木生火，太少交替 |

客运：**太土 → 少金 → 太水 → 少木 → 太火**

## 主运与客运的关系

| 项目 | 主运 | 客运 |
|------|------|------|
| 性质 | 常规之运 | 变化之运 |
| 变化 | 年年固定不变 | 逐年变化 |
| 初运 | 固定从木开始 | 以大运为初运 |
| 意义 | 反映正常气候规律 | 反映异常气候变化 |
| 太少起点 | 由大运所在步定 | 由大运太少定 |

主运为常，客运为变。主运反映一年五季的正常气候变化，客运则在此基础上叠加年度运气变化的影响。两者结合，共同描述一年各时段的气候特征。

## 参考公式

```
五行相生: 木→火→土→金→水→木（循环）
主运固定顺序: [木, 火, 土, 金, 水]
大运五行→主运步位: {木:0, 火:1, 土:2, 金:3, 水:4}

太少交替:
  太过年: 大运步=太, 其余交替
  不及年: 大运步=少, 其余交替

客运排列:
  初运 = 大运五行
  后续 = 五行相生
  太少: 太过从太始, 不及从少始, 交替
```
