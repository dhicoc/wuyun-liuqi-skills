# 太过不及平气

## 太过与不及

### 太过

太过是指阳干年（甲、丙、戊、庚、壬）的运气偏盛。

| 项目 | 说明 |
|------|------|
| 天干 | 甲、丙、戊、庚、壬（阳干） |
| 运气特征 | 该运之气偏盛 |
| 气候偏向 | 偏向该运五行所主气候 |
| 病理倾向 | 该运所主脏腑及所克脏腑易病 |

### 不及

不及是指阴干年（乙、丁、己、辛、癸）的运气偏衰。

| 项目 | 说明 |
|------|------|
| 天干 | 乙、丁、己、辛、癸（阴干） |
| 运气特征 | 该运之气偏衰 |
| 气候偏向 | 偏向所不胜之运（克我之运）的气候 |
| 病理倾向 | 克我之运所主脏腑易病 |

## 平气判断规则

平气之年是指运气虽太过或不及，但经司天之气调节后趋于平和的年份。

### 规则一：太过被抑

太过之运被司天所克 → 平气

判断条件：`司天五行 克 大运五行`（且大运为太过）

示例：戊辰年
- 大运 = 火运太过（戊→火，戊为阳干→太过）
- 司天 = 太阳寒水（辰→太阳寒水）
- 水克火 → 火运太过被抑 → **平气**

### 规则二：不及得助

不及之运得司天同气相助或司天所生 → 平气

此规则包含两种情形：

**情形A：同气相助** — 司天五行与不及大运五行相同，同气相济

判断条件：`司天五行 == 大运五行`（且大运为不及）

示例：丁巳年
- 大运 = 木运不及（丁→木，丁为阴干→不及）
- 司天 = 厥阴风木（巳→厥阴风木）
- 木同气相助 → 木运不及得助 → **平气**

**情形B：司天生运** — 司天五行生不及大运五行，司天助运

判断条件：`司天五行 生 大运五行`（且大运为不及）

示例：癸亥年
- 大运 = 火运不及（癸→火，癸为阴干→不及）
- 司天 = 厥阴风木（亥→厥阴风木）
- 木生火 → 司天生运 → 火运不及得助 → **平气**

### 平气判断汇总

| 大运 | 司天 | 条件 | 结果 |
|------|------|------|------|
| 太过 | 克大运五行 | 司天克运 | 平气（太过被抑） |
| 不及 | 同大运五行 | 司天同气助运 | 平气（同气相助） |
| 不及 | 生大运五行 | 司天生运 | 平气（不及得助） |
| 太过 | 同大运五行 | 司天同气 | 非平气（同化更盛） |
| 不及 | 克大运五行 | 司天克运 | 非平气（不及更衰） |

## 五运太过不及的气候特征

### 太过之年

| 大运 | 太过气候 | 病理倾向 |
|------|----------|----------|
| 木太过 | 风气偏盛 | 肝病，脾病（木克土） |
| 火太过 | 热气偏盛 | 心病，肺病（火克金） |
| 土太过 | 湿气偏盛 | 脾病，肾病（土克水） |
| 金太过 | 燥气偏盛 | 肺病，肝病（金克木） |
| 水太过 | 寒气偏盛 | 肾病，心病（水克火） |

### 不及之年

不及之年，所不胜之运（克我之运）来乘，气候偏向克我之运的特征：

| 大运 | 不及气候 | 克我之运 | 病理倾向 |
|------|----------|----------|----------|
| 木不及 | 燥气偏盛 | 金克木 | 肺病，肝病 |
| 火不及 | 寒气偏盛 | 水克火 | 肾病，心病 |
| 土不及 | 风气偏盛 | 木克土 | 肝病，脾病 |
| 金不及 | 热气偏盛 | 火克金 | 心病，肺病 |
| 水不及 | 湿气偏盛 | 土克水 | 脾病，肾病 |

## 平气之年的特征

- 气候相对平和，不过偏亦不过衰
- 疾病发生相对较少，病情相对较轻
- 运气变化对正常季节气候影响较小

## 参考公式

```
# 太过不及判断
YANG_GAN = ["甲", "丙", "戊", "庚", "壬"]  # 太过
YIN_GAN = ["乙", "丁", "己", "辛", "癸"]    # 不及

# 五行相克
KE = {"木":"土", "土":"水", "水":"火", "火":"金", "金":"木"}    # A克B
SHENG = {"木":"火", "火":"土", "土":"金", "金":"水", "水":"木"}  # A生B

# 平气判断
def pingqi(dayun_wuxing, dayun_type, sitian_wuxing):
    if dayun_type == "太过" and KE[sitian_wuxing] == dayun_wuxing:
        return "平气"  # 规则一：司天克太过之运（太过被抑）
    if dayun_type == "不及" and sitian_wuxing == dayun_wuxing:
        return "平气"  # 规则二A：司天同气助不及之运（同气相助）
    if dayun_type == "不及" and SHENG[sitian_wuxing] == dayun_wuxing:
        return "平气"  # 规则二B：司天生不及之运（司天生运）
    return "非平气"
```
