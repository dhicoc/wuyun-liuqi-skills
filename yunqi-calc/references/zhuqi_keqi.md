# 主气与客气

## 主气

主气是每年固定不变的六步气，反映一年六个时段的正常气候规律。主气按季节顺序排列，年年相同。

### 主气六步

| 步次 | 六气 | 五行 | 起止节气 | 气候特征 |
|------|------|------|----------|----------|
| 初气 | 厥阴风木 | 木 | 大寒→春分 | 风气渐生，春温之始 |
| 二气 | 少阴君火 | 火 | 春分→小满 | 热气渐盛 |
| 三气 | 少阳相火 | 火 | 小满→大暑 | 火气最盛，暑热之极 |
| 四气 | 太阴湿土 | 土 | 大暑→秋分 | 湿气偏盛 |
| 五气 | 阳明燥金 | 金 | 秋分→小雪 | 燥气渐生 |
| 终气 | 太阳寒水 | 水 | 小雪→大寒 | 寒气偏盛 |

主气六步固定排列，反映一年六季的正常气候变化，年年不变。

## 客气

客气是随司天逐年变化的六步气，反映一年六个时段的异常气候变化。客气按阴阳推移顺序排列。

### 客气推移顺序

客气按阴阳级别交替推移：

```
一阴（厥阴风木）→ 二阴（少阴君火）→ 三阴（太阴湿土）→ 一阳（少阳相火）→ 二阳（阳明燥金）→ 三阳（太阳寒水）→ 回到一阴
```

### 客气推移序列

```
KEQI_CYCLE = [厥阴风木, 少阴君火, 太阴湿土, 少阳相火, 阳明燥金, 太阳寒水]
```

## 客气六步排列规则

客气六步的排列以司天和在泉为锚点：

- **司天**为三之气（第三步客气）
- **在泉**为终之气（第六步客气）

其余各步按推移顺序排列。

### 客气推移公式

```
KEQI_CYCLE[(sitian_index + step - 3) % 6]
```

其中：
- `sitian_index`：司天在 KEQI_CYCLE 中的索引（0-5）
- `step`：客气步次（1-6，初气到终气）
- `KEQI_CYCLE`：[厥阴风木, 少阴君火, 太阴湿土, 少阳相火, 阳明燥金, 太阳寒水]

### 推移逻辑说明

- step=3（三之气）时，`KEQI_CYCLE[(sitian_index + 0) % 6]` = 司天本身
- step=6（终之气）时，`KEQI_CYCLE[(sitian_index + 3) % 6]` = 在泉（司天后三位）
- 初气（step=1）为司天前两步：`KEQI_CYCLE[(sitian_index - 2) % 6]`

## 示例：2026丙午年

### 基本推算

- 地支：午 → 司天 = 少阴君火
- 少阴君火在 KEQI_CYCLE 中索引 = 1（二阴）
- 在泉 = 阳明燥金（少阴君火配阳明燥金，二阴配二阳）

### 客气六步排列

| 步次 | 客气 | 推算 | 阴阳 |
|------|------|------|------|
| 初气 | 太阳寒水 | KEQI_CYCLE[(1+1-3)%6] = KEQI_CYCLE[5] | 三阳 |
| 二气 | 厥阴风木 | KEQI_CYCLE[(1+2-3)%6] = KEQI_CYCLE[0] | 一阴 |
| 三气 | 少阴君火 | KEQI_CYCLE[(1+3-3)%6] = KEQI_CYCLE[1] | 二阴（司天） |
| 四气 | 太阴湿土 | KEQI_CYCLE[(1+4-3)%6] = KEQI_CYCLE[2] | 三阴 |
| 五气 | 少阳相火 | KEQI_CYCLE[(1+5-3)%6] = KEQI_CYCLE[3] | 一阳 |
| 终气 | 阳明燥金 | KEQI_CYCLE[(1+6-3)%6] = KEQI_CYCLE[4] | 二阳（在泉） |

**客气排列：太阳寒水 → 厥阴风木 → 少阴君火（司天）→ 太阴湿土 → 少阳相火 → 阳明燥金（在泉）**

### 客气与主气对照

| 步次 | 主气 | 客气 | 备注 |

| 二气 | 少阴君火 | 厥阴风木 | |
| 三气 | 少阳相火 | 少阴君火 | 司天 |
| 四气 | 太阴湿土 | 太阴湿土 | 客主同气 |
| 五气 | 阳明燥金 | 少阳相火 | |

## 参考公式汇总

```
# 客气推移序列
KEQI_CYCLE = ["厥阴风木", "少阴君火", "太阴湿土", "少阳相火", "阳明燥金", "太阳寒水"]

# 司天索引
SITIAN_INDEX = {"厥阴风木":0, "少阴君火":1, "太阴湿土":2, "少阳相火":3, "阳明燥金":4, "太阳寒水":5}

# 客气六步推算
for step in range(1, 7):
    keqi = KEQI_CYCLE[(sitian_index + step - 3) % 6]
```
