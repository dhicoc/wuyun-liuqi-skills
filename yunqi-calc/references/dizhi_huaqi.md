# 地支化六气

## 地支化气原理

地支化六气是地支与六气在运气学说中的对应关系。地支有十二（子丑寅卯辰巳午未申酉戌亥），六气有六（厥阴风木、少阴君火、少阳相火、太阴湿土、阳明燥金、太阳寒水），两地支配一气，两两相合化生六气。

地支化气的原理依据《素问·天元纪大论》所载，十二地支按相隔六位的方式两两配对，每对化生一气，用以确定每年的**司天**之气。

## 司天与在泉的概念

- **司天**：主上半年（初气到大暑）的客气，位于天（上），由地支所化的六气决定
- **在泉**：主下半年（四气到大寒）的客气，位于地（下），与司天阴阳配对

司天与在泉是一对阴阳相对的概念，司天在上主上半年气候，在泉在下主下半年气候。二者构成一年客气变化的总框架。

## 司天-在泉配对（一阴对一阳）

司天与在泉按阴阳级别配对，同级阴阳互配：

| 司天（上） | 阴阳级别 | 在泉（下） | 阴阳级别 |
|------------|----------|------------|----------|
| 厥阴风木 | 一阴 | 少阳相火 | 一阳 |
| 少阴君火 | 二阴 | 阳明燥金 | 二阳 |
| 太阴湿土 | 三阴 | 太阳寒水 | 三阳 |
| 少阳相火 | 一阳 | 厥阴风木 | 一阴 |
| 阳明燥金 | 二阳 | 少阴君火 | 二阴 |
| 太阳寒水 | 三阳 | 太阴湿土 | 三阴 |

配对原则：一阴配一阳，二阴配二阳，三阴配三阳。司天与在泉的阴阳级别相同，但阴阳属性相反。

## 六气的阴阳归类与推移顺序

六气按阴阳多少排列，形成固定的推移顺序：

### 阴阳归类

| 级别 | 阴气 | 阳气 |
|------|------|------|
| 一 | 厥阴风木（一阴） | 少阳相火（一阳） |
| 二 | 少阴君火（二阴） | 阳明燥金（二阳） |
| 三 | 太阴湿土（三阴） | 太阳寒水（三阳） |

### 推移顺序

客气按阴阳交替推移，顺序为：

```
一阴（厥阴）→ 二阴（少阴）→ 三阴（太阴）→ 一阳（少阳）→ 二阳（阳明）→ 三阳（太阳）→ 回到一阴
```

即：厥阴风木 → 少阴君火 → 太阴湿土 → 少阳相火 → 阳明燥金 → 太阳寒水 → 厥阴风木（循环）

此推移顺序是客气六步排列的基础，详见 [主气与客气](zhuqi_keqi.md)。
