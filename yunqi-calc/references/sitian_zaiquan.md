# 司天与在泉

## 概念

- **司天**：主上半年（初气到大暑）的客气主导力量，位于天（上），由该年地支所化的六气决定
- **在泉**：主下半年（四气到大寒）的客气主导力量，位于地（下），与司天阴阳配对

司天在上主上半年气候，在泉在下主下半年气候，二者共同构成一年客气变化的总框架。

## 司天在泉配对原理

司天与在泉按**阴阳级别相同、阴阳属性相反**的原则配对：

| 配对原则 | 说明 |
|----------|------|
| 一阴配一阳 | 厥阴风木 ↔ 少阳相火 |
| 二阴配二阳 | 少阴君火 ↔ 阳明燥金 |
| 三阴配三阳 | 太阴湿土 ↔ 太阳寒水 |

配对逻辑：阴阳级别相同（一、二、三），但一阴配一阳、二阴配二阳、三阴配三阳。司天为阴则在泉为同级别之阳，司天为阳则在泉为同级别之阴。

## 司天在泉对气候的影响

司天主上半年的气候特征，在泉主下半年的气候特征。司天在泉的六气属性决定了该年上下半年的气候偏向。

### 六气司天在泉的气候特点

| 司天/在泉 | 气候特点 | 病理倾向 |
|-----------|----------|----------|
| 厥阴风木 | 风胜 | 肝病、风病、振掉、眩晕 |
| 少阴君火 | 热胜 | 心病、热病、疮疡、出血 |
| 太阴湿土 | 湿胜 | 脾病、湿病、肿胀、泄泻 |
| 少阳相火 | 火胜 | 心包病、火病、发热、疮毒 |
| 阳明燥金 | 燥胜 | 肺病、燥病、干咳、便秘 |
| 太阳寒水 | 寒胜 | 肾病、寒病、疼痛、收引 |

### 司天主上半年

司天之气主导初气至三气（大寒至大暑）的气候特征。如太阳寒水司天，上半年偏寒；少阳相火司天，上半年偏热。

### 在泉主下半年

在泉之气主导四气至终气（大暑至大寒）的气候特征。如太阴湿土在泉，下半年偏湿；厥阴风木在泉，下半年多风。
