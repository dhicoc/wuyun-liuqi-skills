# 六十甲子表

## 一、六十甲子的排列规则

六十甲子由天干与地支按"阳干配阳支、阴干配阴支"的原则同步递进排列而成。天干从甲开始，地支从子开始，各自依次递进，每推进一位组成一个干支。因天干10位、地支12位，最小公倍数为60，故60组后循环回归，称为"六十甲子"。

排列法则：
1. 天干顺序：甲→乙→丙→丁→戊→己→庚→辛→壬→癸→甲……（10位循环）
2. 地支顺序：子→丑→寅→卯→辰→巳→午→未→申→酉→戌→亥→子……（12位循环）
3. 同步推进：天干进一位，地支同时进一位
4. 60年一循环：从甲子（序号1）到癸亥（序号60），周而复始

## 三、六十甲子与五运六气的关系

### 大运（中运）规律

每个甲子年的大运由天干决定，60年中共有6个完整的五运循环（10年×6=60年）：

| 天干 | 化运 | 出现次数 | 太过/不及 |
|------|------|---------|-----------|
| 甲 | 土运 | 6次 | 太过（阳干） |
| 乙 | 金运 | 6次 | 不及（阴干） |
| 丙 | 水运 | 6次 | 太过（阳干） |
| 丁 | 木运 | 6次 | 不及（阴干） |
| 戊 | 火运 | 6次 | 太过（阳干） |
| 己 | 土运 | 6次 | 不及（阴干） |
| 庚 | 金运 | 6次 | 太过（阳干） |
| 辛 | 水运 | 6次 | 不及（阴干） |
| 壬 | 木运 | 6次 | 太过（阳干） |
| 癸 | 火运 | 6次 | 不及（阴干） |

> 六十甲子中，五运各出现12次（太过6次+不及6次）。

### 司天在泉规律

每个甲子年的司天由地支决定，60年中共有5个完整的六气循环（12年×5=60年）：

| 地支 | 司天 | 在泉 | 出现次数 |
|------|------|------|---------|
| 子、午 | 少阴君火 | 阳明燥金 | 各5次，共10次 |
| 丑、未 | 太阴湿土 | 太阳寒水 | 各5次，共10次 |
| 寅、申 | 少阳相火 | 厥阴风木 | 各5次，共10次 |
| 卯、酉 | 阳明燥金 | 少阴君火 | 各5次，共10次 |
| 辰、戌 | 太阳寒水 | 太阴湿土 | 各5次，共10次 |
| 巳、亥 | 厥阴风木 | 少阳相火 | 各5次，共10次 |

### 六十甲子运气总格局

六十甲子可按六气分为六组，每组10年，司天相同：

| 组别 | 地支 | 司天 | 在泉 | 对应干支序号 |
|------|------|------|------|-------------|
| 第一组 | 子、午 | 少阴君火 | 阳明燥金 | 甲子~癸酉（1-10）、甲午~癸卯（31-40） |
| 第二组 | 丑、未 | 太阴湿土 | 太阳寒水 | 甲戌~癸未（11-20后段实际为丑未年） |
| 第三组 | 寅、申 | 少阳相火 | 厥阴风木 | 含寅申年 |
| 第四组 | 卯、酉 | 阳明燥金 | 少阴君火 | 含卯酉年 |
| 第五组 | 辰、戌 | 太阳寒水 | 太阴湿土 | 含辰戌年 |
| 第六组 | 巳、亥 | 厥阴风木 | 少阳相火 | 含巳亥年 |

> 每组的10年中，五运太过与不及各5年，形成固定的运气组合模式。
