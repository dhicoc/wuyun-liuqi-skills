# 运气合病参考文档

> 本文档为 `yunqi-pathogenesis` 子技能的病机分析参考，专论大运与六气共同影响下的复合病机。
> 前置条件：须先从 `yunqi-calc` 获取大运及六气推算结果，并完成五运病机、六气病机、太过不及病机的初步分析。

---

## 一、运气合病的概念

### 1.1 定义

运气合病，指大运（中运）与六气（司天在泉、客气加临）共同作用于人体，形成的复合病机。

- 大运定年度总基调，决定脏腑偏盛偏衰的总体方向
- 六气定上下半年及各步时令的病机变化
- 二者叠加，形成运气合病的复合病机

### 1.2 分析层次

1. **大运** → 定位脏腑（如木运→肝，火运→心）
2. **六气** → 定位经络（如厥阴风木→肝经，少阴君火→心经）
3. **客气加临** → 定时令（如初之气厥阴→春季发病趋势）
4. **运气关系** → 定病机轻重与顺逆（同化、不和、顺化、天刑）

## 二、运气同化年的病机特点

运气同化，指大运与六气五行属性相同或相合，导致气候偏盛，对应脏腑易病。

### 2.1 天符年

- **定义**：大运与司天之气五行属性相同
- **判断**：如乙年金运，司天为阳明燥金 → 运与司天同为金 → 天符
- **病机特点**：运气同气，气候偏盛，对应脏腑极易发病
  - 该年该脏腑系统病变尤为突出
  - 疾病较一般年份偏重
- **经典依据**：《素问·六微旨大论》"天符为执法"

### 2.2 岁会年

- **定义**：大运与岁支（地支）五行属性相同
- **判断**：如丁年木运，岁支为卯（东方木）→ 运与支同气 → 岁会
- **病机特点**：运气与岁支同气，气候偏于平和或偏盛
  - 岁会之年气候相对平稳（得岁支之正）
  - 但仍需结合司天在泉综合判断
- **经典依据**：《素问·六微旨大论》"岁会为行令"

### 2.3 太乙天符

- **定义**：大运、司天之气、岁支三者五行属性皆同
- **判断**：如己年土运，岁支为丑（土），司天为太阴湿土 → 三者同气 → 太乙天符
- **病机特点**：三者同气，气候极偏，疾病最重
  - 该年该脏腑系统病变极为突出
  - 疾病最为严重，且变化迅速
- **经典依据**：《素问·六微旨大论》"太乙天符为贵人…邪之中人也，中执法者，其病速而危；中行令者，其病徐而持；中贵人者，其病暴而死"

### 2.4 同化年病机比较

| 同化类型 | 条件 | 偏盛程度 | 疾病轻重 |
|----------|------|----------|----------|
| 天符 | 运与司天同气 | 偏盛 | 较重，病速而危 |
| 岁会 | 运与岁支同气 | 较平和 | 较轻，病徐而持 |
| 太乙天符 | 运、司天、岁支皆同 | 极偏 | 最重，病暴而死 |

## 三、运气不和（运克气/气克运）的病机

### 3.1 运克气

- **定义**：大运之气克司天之气（如土运克水司天）
- **病机特点**：
  - 运气不和，气候乖戾
  - 大运偏盛，克伐六气，六气受抑
  - 病机以大运所主脏腑偏盛为主，六气所主脏腑受抑
- **分析**：运胜则运之所胜受邪，气被克则气之所主脏腑功能受抑

### 3.2 气克运

- **定义**：司天之气克大运（如水司天克火运）
- **病机特点**：
  - 运气不和，气候逆乱
  - 六气偏盛，克伐大运，大运受抑
  - 病机以六气所主脏腑偏盛为主，大运所主脏腑受抑
- **分析**：气胜则气之所胜受邪，运被克则运之所主脏腑功能受抑

## 四、顺化与天刑的病机

### 4.1 顺化（运生气）

- **定义**：大运之气生司天之气（如木运生火司天）
- **病机特点**：
  - 运气相生为顺，气候较为调和
  - 运为气之母，母子相生，气得运之助
  - 病机相对较轻，但仍需结合太过不及判断
- **分析**：顺化之年，运助气盛，气所胜之脏腑可能受邪

### 4.2 天刑（气克运）

- **定义**：司天之气克大运（即上述气克运的情况，又称天刑）
- **病机特点**：
  - 气克运为逆，气候偏逆
  - 司天之气偏盛，克伐大运，大运受抑
  - 若大运太过被司天克之，可转为平气（太过被抑）
  - 若大运不及被司天克之，则不及更衰，病机加重
- **分析**：天刑之年，气胜运衰，病机以气所主脏腑偏盛为主

### 4.3 顺化与天刑比较

| 类型 | 关系 | 顺逆 | 病机轻重 |
|------|------|------|----------|
| 顺化 | 运生气 | 顺 | 较轻 |
| 天刑 | 气克运 | 逆 | 较重（不及之年）或转平（太过之年） |

## 五、病机十九条（与六气相关条文）

## 六、运气合病的分析框架

### 6.1 三维定位法

```
大运 → 定位脏腑（确定年度总体脏腑发病方向）
六气 → 定位经络（确定上下半年及各步时令的经络病变）
客气加临 → 定时令（确定具体时段的病机变化）
```

### 6.2 分析步骤

1. **大运定位脏腑**：
   - 确定大运五行属性（木火土金水）
   - 确定太过不及
   - 定位本脏及所胜/所不胜脏腑

2. **六气定位经络**：
   - 确定司天之气 → 上半年经络病变方向
   - 确定在泉之气 → 下半年经络病变方向
   - 司天在泉与脏腑对应关系

3. **客气加临定时令**：
   - 六步客气加临主气
   - 判断客主相得/不相得
   - 确定各步时令的病机变化

4. **运气关系定轻重**：
   - 判断运气同化（天符/岁会/太乙天符）
   - 判断运气不和（运克气/气克运）
   - 判断顺化/天刑
   - 综合确定病机轻重与顺逆

### 6.3 分析示例

**示例：木运太过 + 厥阴风木司天**

- 大运：木太过 → 肝气偏盛，脾土受邪，金气来复
- 司天：厥阴风木 → 上半年风证多发，肝系病变偏盛
- 运气关系：运与司天同气 → 天符年 → 偏盛极重
- 综合判断：
  - 肝系病变最为突出（运气同气，偏盛极重）
  - 脾土受邪明显（木克土）
  - 上半年风证为主，下半年视在泉之气而定
  - 病机十九条验证："诸风掉眩，皆属于肝"

### 6.4 运气合病分析总表

| 运气关系 | 特点 | 病机轻重 | 对应脏腑 |
|----------|------|----------|----------|
| 天符 | 运与司天同气 | 偏盛，较重 | 运/气同主脏腑 |
| 岁会 | 运与岁支同气 | 较平和 | 运主脏腑 |
| 太乙天符 | 三者同气 | 极偏，最重 | 运/气/支同主脏腑 |
| 运克气 | 运胜气 | 运主脏腑偏盛 | 运主脏腑 |
| 气克运 | 气胜运 | 气主脏腑偏盛 | 气主脏腑 |
| 顺化 | 运生气 | 较轻 | 气主脏腑（得运之助） |
| 天刑 | 气克运 | 较重 | 运主脏腑（受气之克） |
