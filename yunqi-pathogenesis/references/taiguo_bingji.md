# 太过不及病机参考文档

> 本文档为 `yunqi-pathogenesis` 子技能的病机分析参考，专论五运太过不及的病机特点及胜复规律。
> 前置条件：须先从 `yunqi-calc` 获取大运推算结果（太过/不及/平气判断）。

## 一、太过之年的病机特点

### 1.1 总则

太过之年，本气偏盛，其病机遵循以下规律：

1. **本气偏盛**：主岁之运太过，则该运所对应脏腑之气偏盛
2. **所胜受邪**：本气盛则乘其所胜（我克者），所胜之脏腑受邪发病
3. **所不胜来复**：本气太过则所不胜（克我者）之气来复，以制其太过

### 1.2 例析

**木太过**：
- 肝气偏盛（本气盛）
- 脾土受邪（木克土，所胜受邪）—— 腹胀、泄泻
- 金气来复（金克木，所不胜来复）—— 肺气偏亢，咳嗽

> 木盛克土，土气受抑；土为金之母，土虚则金气来复以制木。
> 此即"亢则害，承乃制"之理。

### 1.3 太过之年病机链条

```
本气偏盛 → 乘所胜（我克者受邪）→ 所不胜（克我者）来复
```

## 二、不及之年的病机特点

### 2.1 总则

不及之年，本气偏衰，其病机遵循以下规律：

1. **本气偏衰**：主岁之运不及，则该运所对应脏腑之气不足
2. **所不胜来乘**：本气衰则所不胜（克我者）之气来乘，本脏受邪
3. **所胜反侮**：本气不及则所胜（我克者）之气反侮，病机更趋复杂

### 2.2 例析

**木不及**：
- 肝气不足（本气衰）
- 肺金来乘（金克木，所不胜来乘）—— 咳嗽、气短
- 土气反侮（土本被木克，木衰则土反侮）—— 腹胀、食少

> 木衰则金乘，金乘则木更衰；木衰不能制土，土无所制则反侮。
> 此即"不足则所不胜乘之，所胜侮之"之理。

### 2.3 不及之年病机链条

```
本气偏衰 → 所不胜（克我者）来乘 → 所胜（我克者）反侮
```

## 三、"胜复"规律

### 3.1 胜复的概念

- **胜**：一气偏盛，克犯他气（如木胜则克土）
- **复**：被克之气来报复，制其偏盛（如木胜则金来复以克木）

### 3.2 胜复总则

| 情形 | 胜 | 复 |
|------|----|----|
| 太过之年 | 本气胜，乘所胜 | 所不胜来复 |
| 不及之年 | 所不胜胜，乘本气 | 所胜反侮（间或本气之子来复） |

### 3.3 胜复规律详解

**太过之年的胜复**：
- 太过则有所不胜来复
- 例：木太过 → 木胜（克土）→ 金来复（金克木）
- 复气即所不胜之气，制其太过，使归于平

**不及之年的胜复**：
- 不及则有所不胜来乘
- 例：木不及 → 金胜（金克木）→ 土反侮（土本被木克，木衰则土反侮）
- 间或本气之子来复：木之子为火，火来复以制金（子复母仇）

### 3.4 胜复的时序

- 胜气在前，复气在后
- 胜气盛于当年，复气可于当年或次年显现
- 胜复交替，终归于平气

## 四、平气之年的病机

### 4.1 平气的判断

平气之年指五运既非太过也非不及，气候相对平和。判断方式包括：

1. **运与岁支同气**：如丁年木运，丁壬化木，岁支为卯木，运与支同气则平
2. **运与司天同气**：运与司天之气相生相合，可得平气
3. **太过而被抑**：太过之运遇司天之气克之，则太过被抑为平气
4. **不及而得助**：不及之运遇司天之气生之，则不及得助为平气

### 4.2 平气之年的病机特点

- 气候平和，病机较轻
- 脏腑发病趋势不甚偏颇
- 但仍需结合六气（司天在泉、客主加临）综合分析
- 不可因平气而忽略客气加临的病机影响

## 六、太过不及与六气的综合判断

> 太过不及的分析须结合六气（司天在泉）综合判断，不可孤立看待。
> 详见 `yunqi_hebing.md` 运气合病篇。

### 6.1 太过遇司天

- 太过之运遇司天同气：偏盛更甚，病机更重
- 太过之运遇司天克之：太过被抑，可转为平气
- 太过之运遇司天生之：太过得助，偏盛更甚

### 6.2 不及遇司天

- 不及之运遇司天同气：得助可转为平气
- 不及之运遇司天克之：不及更衰，病机加重
- 不及之运遇司天生之：得助可转为平气

### 6.3 综合判断原则

- 运气同化（天符、岁会、太乙天符）：偏盛极重
- 运气不和（运克气/气克运）：病机复杂
- 顺化（运生气）：气候较和
- 天刑（气克运）：气候偏逆

> 以上综合判断须结合 `yunqi_hebing.md` 完成。
