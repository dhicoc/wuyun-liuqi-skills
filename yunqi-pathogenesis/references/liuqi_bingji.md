# 六气病机参考文档

> 本文档为 `yunqi-pathogenesis` 子技能的病机分析参考，按厥阴、少阴、少阳、太阴、阳明、太阳六气分别论述发病规律。
> 前置条件：须先从 `yunqi-calc` 获取六气推算结果（司天、在泉、客主加临）。

## 一、六气各自发病规律

### 1.1 厥阴风木

- **对应脏腑**：肝系
- **病证性质**：风证
- **主要症状**：眩晕、震颤、拘急、瘙痒
- **病机阐述**：
  - 厥阴风木主肝，风气通于肝
  - 风为阳邪，善行数变，易侵犯人体上部、肌表
  - 风气偏盛则肝气横逆，见眩晕（肝风内动）、震颤（风性动摇）
  - 风客肌表则见拘急（风性收引）、瘙痒（风邪搏于肌肤）
- **经典依据**：《素问·至真要大论》"厥阴之胜，耳鸣头眩，愦愦欲吐"

### 1.2 少阴君火

- **对应脏腑**：心系
- **病证性质**：热证
- **主要症状**：心烦、失眠、口疮、出血
- **病机阐述**：
  - 少阴君火主心，热气通于心
  - 君火偏盛则心火亢，心主神明，火扰神明则心烦、失眠
  - 心开窍于舌，火上炎则口疮
  - 火迫血妄行则见各种出血（鼻衄、齿衄、肌衄等）
- **经典依据**：《素问·至真要大论》"少阴之胜，心下热善饥，脐下反痛"

### 1.3 少阳相火

- **对应脏腑**：三焦系（兼胆）
- **病证性质**：火证
- **主要症状**：发热、咽痛、耳鸣、往来寒热
- **病机阐述**：
  - 少阳相火主三焦，相火寄于肝胆
  - 相火偏盛则三焦火盛，三焦为水火通行之道路，火盛则水道不利
  - 相火上炎则见发热、咽痛、耳鸣（少阳经入耳）
  - 少阳为枢，枢机不利则往来寒热（正邪交争于半表半里）
- **经典依据**：《素问·至真要大论》"少阳之胜，暴注瞤瘛，暴病暴死"

### 1.4 太阴湿土

- **对应脏腑**：脾系
- **病证性质**：湿证
- **主要症状**：腹胀、泄泻、水肿、身重
- **病机阐述**：
  - 太阴湿土主脾，湿气通于脾
  - 湿为阴邪，易困阻中焦，脾失健运
  - 湿困脾土则见腹胀、泄泻（脾主运化，湿困则运化失司）
  - 湿溢肌肤则见水肿、身重（湿性重浊黏腻）
- **经典依据**：《素问·至真要大论》"太阴之胜，火气内郁，疮疡于中，散于胠胁"

### 1.5 阳明燥金

- **对应脏腑**：肺系（兼大肠）
- **病证性质**：燥证
- **主要症状**：咳嗽、皮肤干、便秘、毛枯
- **病机阐述**：
  - 阳明燥金主肺，燥气通于肺
  - 燥为阳邪，易伤津液，肺主皮毛，燥胜则皮毛失养
  - 燥邪犯肺则见咳嗽（肺气上逆）、皮肤干（津伤皮失所养）
  - 燥结大肠则便秘（大肠主传导，燥结则传导失司）
  - 肺合皮毛，毛为肺之外候，燥胜则毛枯
- **经典依据**：《素问·至真要大论》"阳明之胜，清发于中，左胠胁痛"

### 1.6 太阳寒水

- **对应脏腑**：肾系（兼膀胱）
- **病证性质**：寒证
- **主要症状**：畏寒、腰痛、小便清长、关节冷痛
- **病机阐述**：
  - 太阳寒水主肾，寒气通于肾
  - 寒为阴邪，易伤阳气，肾为寒水之脏，寒盛则肾阳受遏
  - 寒盛阳虚则见畏寒（阳虚不温）
  - 腰为肾之府，寒凝腰府则腰痛
  - 寒盛则肾不化气，小便清长（寒则气化不利，水液下趋）
  - 寒凝关节则关节冷痛（寒性收引凝滞）
- **经典依据**：《素问·至真要大论》"太阳之胜，凝溧且至，寒厥入胃"

## 三、客气加临主气时的病机变化

### 3.1 六步主气与客气加临

一年分为六步，每步约60日，主气固定不变，客气逐年流转：

| 步次 | 主气 | 时令 | 客气加临后可能的变化 |
|------|------|------|----------------------|
| 初之气 | 厥阴风木 | 大寒至春分 | 客气加临后，若客气为风则风盛，为寒则春行冬令，为热则温病早发 |
| 二之气 | 少阴君火 | 春分至小满 | 客气为热则热盛，为湿则湿热交蒸，为寒则倒春寒 |
| 三之气 | 少阳相火 | 小满至大暑 | 客气为火则暑热盛，为燥则旱热，为寒则夏行秋令 |
| 四之气 | 太阴湿土 | 大暑至秋分 | 客气为湿则湿盛，为燥则长夏行秋令，为寒则湿中兼寒 |
| 五之气 | 阳明燥金 | 秋分至小雪 | 客气为燥则燥盛，为热则秋行夏令，为湿则秋雨连绵 |
| 终之气 | 太阳寒水 | 小雪至大寒 | 客气为寒则寒盛，为热则冬行夏令（蛰虫不藏），为风则寒中兼风 |

### 3.2 客主加临的相得与不相得

- **客主相得（顺）**：客气与主气五行相同或相生，气候平和，病机较轻
  - 例：初之气主气厥阴风木，客气亦厥阴风木 → 气同，风盛但仍属顺
  - 例：初之气主气厥阴风木，客气少阴君火 → 木生火，相生为顺
- **客主不相得（逆）**：客气与主气五行相克，气候反常，病机较重
  - 例：初之气主气厥阴风木，客气阳明燥金 → 金克木，相克为逆
  - 例：初之气主气厥阴风木，客气太阳寒水 → 水生木为相生，但寒水使春令推迟

### 3.3 客主加临与疾病时令

- 客主相得之年：疾病较轻，按常规季节性发病
- 客主不相得之年：疾病较重，非时之气导致反季节发病
  - 如春行冬令（初之气客气为寒）→ 春季多发寒病
  - 如夏行秋令（三之气客气为燥）→ 夏季多发燥病
- 司天之客气加临三之气（少阳相火），决定上半年气候主流
- 在泉之客气加临终之气（太阳寒水），决定下半年气候主流
