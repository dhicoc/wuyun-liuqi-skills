# 示例案例库

本目录存放脱敏后的五运六气分析示例，用于展示项目的标准推理链路与输出格式。

## 示例列表

| 文件 | 类型 | 说明 |
|------|------|------|
| `case-2003-birth-2026-weather-constitution.md` | 个人高级对齐示例 | 出生日期 2003-04-19，分析日期 2026-06-29，杭州，mock 天气与九种体质示例 |
| `case-2026-practitioner-report.md` | 年度临床版报告示例 | 2026 丙午年，含经典与注家依据、安全提示 |

## 脱敏与安全规则

1. 示例不得包含真实姓名、电话、地址、身份证、病历号等个人敏感信息。
2. 示例仅用于说明五运六气理论推演流程，不构成诊疗建议。
3. 涉及方药、针灸、穴位时必须保留免责声明和执业操作提示。
4. 示例中使用固定日期与 mock 天气数据，避免外部 API 漂移。
