# 经典文献（运气学说文献检索与理论溯源）

## 适用范围

- 研究者：运气学说理论溯源与学术研究
- 素问七篇大论内容检索与篇目定位
- 历代运气学说各家观点查阅（唐至清代）
- 现代运气学说研究概况了解
- 运气推算结果的理论依据验证
- 为下游报告生成提供文献引用支撑

## 脚本依赖

本子技能为纯文献检索，无脚本依赖。但需调用上游 `ganzhi-basics` 的推算结果进行理论验证。

| 依赖类型 | 来源 | 说明 |
|---------|------|------|
| 数据依赖（上游） | `ganzhi-basics` | 干支推算结果、大运、六气格局 |
| 参考文件 | `references/suwen_qipian.md` | 素问七篇大论索引 |
| 参考文件 | `references/lidai_xueshuo.md` | 历代运气学说 |
| 参考文件 | `references/xiandai_yanjiu.md` | 现代研究文献 |

## 推荐工作流

1. **ACT: 接收上游推算结果**，从 `ganzhi-basics` 获取干支、大运、六气格局等信息。明确需要验证或溯源的理论问题。
2. **ACT: 确定文献范围**，根据研究目的选择检索范围：
   - 经典溯源 → 查阅 `references/suwen_qipian.md`（素问七篇大论）
   - 历代学术 → 查阅 `references/lidai_xueshuo.md`（历代各家学说）
   - 现代研究 → 查阅 `references/xiandai_yanjiu.md`（现代研究概况）
3. **ACT: 检索对应文献**，在选定的参考文件中定位与推算结果相关的理论论述。例如：大运太过/不及的理论依据→《气交变大论》；六气主客加临→《六元正纪大论》；病机治法→《至真要大论》。
4. **ACT: 整理引用**，将检索到的文献内容整理为结构化引用，包含：篇名、篇号、核心论述、与推算结果的关联。
5. **ACT: 关联推算验证**，将文献理论论述与上游推算结果对照，验证推算的理论一致性。如发现矛盾，标注差异并说明可能原因。
6. **ACT: 汇总输出**，将文献检索结果、引用信息、验证结论整理为结构化输出，传递给 `docs-generator` 用于报告生成。

## 常见误区

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 引用篇号错误 | 素问七篇大论篇号与通行本《黄帝内经·素问》编排不一致 | 严格参照 `references/suwen_qipian.md` 中的篇号索引。七篇大论为第66-71篇及第74篇 |
| 历代学说混淆 | 各家运气学说观点存在差异，误将一家之言当作标准 | 查阅 `references/lidai_xueshuo.md`，明确各家观点的来源与特点，引用时标注出处 |
| 理论与实践脱节 | 仅引用经典原文，未与具体推算结果关联 | Step5必须执行：将文献论述与推算结果对照验证，不可跳过 |
| 现代研究引用不当 | 将现代研究的"研究方向概述"误当作"具体论文结论" | `references/xiandai_yanjiu.md` 仅为研究方向概述，不含具体论文引用。如需引用具体论文，须另行检索学术数据库 |
| 运气七篇归属错误 | 误将七篇大论当作素问原有内容 | 七篇大论由王冰（唐代）整理补入素问，其来源与成书年代有学术争议，引用时须注明 |

## 输出要求

- 文献引用须包含：篇名、篇号（如适用）、核心论述摘要
- 须区分经典原文、历代注家观点、现代研究方向三类来源
- 须标注引用出处（参考文件名+章节）
- 验证结论须明确：一致/部分一致/存在差异，并说明依据
- 所有术语须使用标准中医运气学说术语
- 不得将研究方向概述当作具体论文引用

## 路由上下文

- **同级关联**：与 `ganzhi-basics` 同级；`ganzhi-basics` 提供推算数据，本子技能提供理论文献支撑
