# 首次使用引导 / 模糊意图回复模板

当用户仅发送触发词（如"五运六气""运气"）或意图不明确时，使用本模板进行轻量级引导，避免直接输出完整报告。

## 使用条件

满足以下任一条件时使用本模板：

- 用户输入仅包含触发词，无具体日期/年份
- 用户未说明需求类型（推算、病机、临床、学习、个人体质）
- 用户输入无法匹配 `routing.md` 中的任何路由项
- 用户说"帮我看看""分析一下""介绍一下"等模糊表达

## 引导回复模板

```
你好，我是五运六气分析助手。我可以帮你：

1. 快速了解当前/指定日期的运气概况
2. 生成完整年度分析报告（学生版 / 临床版 / 研究版）
3. 分析当前步位的运气病机
4. 提供运气治法、方药方向或养生建议
5. 学习运气学基础概念（天干化五运、地支化六气、客主加临等）
6. 分析个人先天运气体质（需提供出生日期和所在地区）

请告诉我：
- 你想分析哪个日期或年份？（可直接说"今天"）
- 你主要关注哪方面？（回复数字 1-6 即可）
```

## 用户常见回复的快速路由

| 用户回复 | 下一步动作 |
|----------|-----------|
| "今天" / "2026-06-29" / 一个日期 | 调用 `python scripts/calculate_yunqi_api.py <日期> --summary` |
| "最近" / "我现在" | 调用 `python scripts/calculate_yunqi_api.py <今天日期> --focus current-step` |
| "1" / "快速了解" | 调用 `--summary`，聚焦当前或默认日期 |
| "2" / "完整分析" | 询问受众（学生/医师/研究者），调用对应 `--report-type` |
| "3" / "病机" | 调用 `calculate_yunqi_api.py` + 读取 `yunqi-pathogenesis/references/` |
| "4" / "治法" / "养生" | 调用 `calculate_yunqi_api.py` + 读取 `yunqi-clinical/references/` |
| "5" / "学习" | 询问具体概念，读取对应 `references/` |
| "6" / "个人体质" | 询问出生日期和地区，调用 `scripts/personal_yunqi_profile.py` |

## 注意事项

- 引导回复保持简洁，不超过 200 字
- 不输出医学建议，除非用户明确进入病机/临床流程
- 用户明确意图后，立即按 `routing.md` 执行，不再重复引导
- 所有临床输出 MUST 附加 `case-journal/precedent-disclaimer.md` 中的免责声明
