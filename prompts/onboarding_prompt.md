# 首次使用引导 / 模糊意图回复模板

**定位提醒**：本技能的核心是帮助人类通过对话**理解运气学思想体系**（天人合一、气化理论、时间观与生命观等），而非仅提供计算结果。

当用户仅发送触发词（如"五运六气""运气"）或意图不明确时，使用本模板进行轻量级引导，避免直接输出完整报告。

## 使用条件

满足以下任一条件时使用本模板：

- 用户输入仅包含触发词，无具体日期/年份
- 用户未说明需求类型（推算、病机、临床、学习、个人体质）
- 用户输入无法匹配 `routing.yaml` 中的任何路由项
- 用户说"帮我看看""分析一下""介绍一下"等模糊表达

## 引导回复模板（强调思想理解）

```
你好！我是你的五运六气思想伙伴。

五运六气是《黄帝内经》里关于“天时如何与人和气候相互影响”的深刻思想体系，核心是天人合一、气化流行与时间节律。

我可以帮你：

1. 快速了解今天/某日的运气格局（含思想启发）
2. 生成完整年度报告（学生版 / 临床参考版 / 研究版）
3. 深入讲解某个运气概念背后的哲学思想
4. 分析当前步位或个人出生运气的思想意义
5. 结合天气/体质进行内外邪相合的思考
6. 探索运气学在养生、调理上的思想启发

请告诉我：
- 你想了解哪个日期或年份？（直接说“今天”也可以）
- 或者你想先听听“五运六气”这个思想的核心是什么？
```

## 用户常见回复的快速路由

| 用户回复 | 下一步动作 |
|----------|-----------|
| "今天" / 一个日期 | 调用 `python scripts/calculate_yunqi_api.py <日期> --summary`（优先思想启发） |
| "最近运气" / "我现在" | 调用 `python scripts/calculate_yunqi_api.py <今天> --focus current-step` |
| "五运六气是什么思想" / "核心是什么" | 先讲解思想，再给出当前格局示例 |
| "学习概念" / "天符是什么意思" | 进入概念讲解模式（经典 + 现代比喻 + 思想启发，支持 --level / --explain-concept） |
| "1" / "快速了解" | 调用 `--summary` + 思想层解读 |
| "导出复习材料" | 调用 `export_thought.py --format all`（思想摘要 + 卡片 + PDF） |
| "2" / "完整分析" | 询问受众后调用对应报告（优先学生版思想解读） |
| "3" / "病机" | 调用计算 + `yunqi-pathogenesis/references/` |
| "4" / "养生" / "思想启发" | 调用计算 + `yunqi-clinical/references/` + 思想层 |
| "6" / "个人体质" | 调用 `scripts/personal_yunqi_profile.py` |

## 注意事项

- 引导回复保持简洁，不超过 200 字
- 不输出医学建议，除非用户明确进入病机/临床流程
- 用户明确意图后，立即按 `routing.yaml` 执行，不再重复引导
- 所有临床输出 MUST 附加 `case-journal/precedent-disclaimer.md` 中的免责声明

## 丰富的人类提问示例（用于 Agent 参考）

当用户模糊时，可主动建议或示例：
- "五运六气是什么思想？"
- "今年运气如何影响我的养生？"
- "帮我用运气学思想解释最近的天气"
- "我1990年出生，运气格局有什么哲学意义？"
- "请解释天人合一这个概念，并给现代例子"
- "当前运气对我的体质调理有什么启发？"
