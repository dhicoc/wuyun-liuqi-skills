## 1. 设计理念：先天运气缺陷 × 后天当下体质

### 1.1 核心思想

运气学说以天干地支推算年度气运偏胜，揭示的是**群体层面的气候-疾病趋势**——某一运气格局下，人群整体易受何种邪气侵袭、何脏腑易病。然而同一运气格局下，不同个体发病与否、发病后的证候特征千差万别，其根本原因在于**个体体质差异**。

中医体质学说认为，体质是个体在先天禀赋与后天获得的基础上形成的，在形态结构、生理机能、心理状态等方面的综合特征。体质决定了人体对病邪的**易感性**与发病后的**证候倾向性**。正如《素问·经脉别论》所言："勇者气行则已，怯者则著而为病也。"同气同邪，勇怯不同，病与不病迥异。

将运气推算与个体体质对齐，体现的是"**先天运气缺陷 + 后天当下体质 = 交叉对比分析**"的精准化思路：

- **先天运气缺陷**：出生年份的运气格局所提示的先天体质倾向（如水运太过之年出生者，先天寒邪偏盛，肾系易虚）
- **后天当下体质**：通过王琦教授九种体质分类量表评估的当下体质状态
- **交叉对比**：将先天运气倾向与后天体质状态交叉分析，判断二者是否叠加、抵消或转化，从而给出更有针对性的调理方向

### 1.2 理论依据

| 来源 | 原文 | 释义 |
|------|------|------|
| 《素问·经脉别论》 | "勇者气行则已，怯者则著而为病也" | 体质决定发病与否 |
| 《素问·阴阳应象大论》 | "阳胜则热，阴胜则寒" | 体质偏胜决定证候倾向 |
| 《灵枢·五变》 | "五藏皆柔弱者，善病消瘅" | 脏腑柔弱（体质偏颇）则易患特定疾病 |
| 王琦《中医体质分类与判定》 | 九种体质分类标准 | 体质可量化评估，且与发病倾向密切相关 |
| 《素问·六微旨大论》 | "上下之位，气交之中，人之居也" | 人处天地气交之中，运气与体质共同作用 |

### 1.3 设计目标

```
先天运气缺陷（出生年运）  +  后天当下体质（问卷评估）  =  交叉对比分析
       ↓                           ↓                           ↓
  先天体质倾向提示           当下体质状态量化           易感性 + 证候倾向
  (birth year yunqi)         (constitution scores)      (cross analysis)
```

通过体质对齐，Agent 可以实现：

1. **个体化病机预判**：从"某年人群易发某病"升级为"此体质者在此运气下易发某证"
2. **调理方向精准化**：结合体质偏颇确定调理的侧重点（如阳虚质+寒水司天→温阳为主）
3. **先天-后天综合评估**：评估先天运气倾向与后天体质是否一致（叠加则风险倍增，相抵则风险降低）
4. **养生方案个体化**：从通用运气养生升级为"体质 × 运气"个体化养生方案

## 2. 中医九种体质（王琦教授分类）

### 2.1 九种体质概览

依据北京中医药大学王琦教授提出的中医体质分类标准（《中医体质分类与判定》ZYYXH/T157-2009），将人体体质分为以下九种基本类型：

| 序号 | 体质类型 | 核心特征 | 易感邪气 | 易病脏腑/系统 | 舌脉特征 |
|------|----------|----------|----------|--------------|----------|
| 1 | **平和质** | 阴阳气血调和，体态适中，面色红润，精力充沛，适应力强 | 无明显易感邪气 | 无明显偏颇 | 舌色淡红，苔薄白，脉和缓 |
| 2 | **气虚质** | 元气不足，气短懒言，易疲乏，自汗，易感冒 | 风邪、寒邪（卫表不固） | 肺、脾（气虚） | 舌淡红，舌体胖大，边有齿痕，脉虚缓 |
| 3 | **阳虚质** | 阳气不足，畏寒怕冷，手足不温，喜温饮食，大便溏薄 | 寒邪、湿邪（阴邪） | 脾、肾（阳虚） | 舌淡胖嫩，苔润，脉沉迟 |
| 4 | **阴虚质** | 阴液亏少，口燥咽干，手足心热，喜冷饮，大便干燥 | 燥邪、火邪（阳邪） | 肝、肾（阴虚） | 舌红少津，苔少或无苔，脉细数 |
| 5 | **痰湿质** | 痰湿凝聚，体形肥胖，腹部肥满，口黏苔腻，喜食肥甘 | 湿邪（内外合邪） | 脾、肺（痰湿困阻） | 舌体胖大，苔白腻，脉滑 |
| 6 | **湿热质** | 湿热内蕴，面垢油光，口苦口干，身重困倦，大便黏滞 | 湿邪、火邪（湿热合邪） | 脾、肝（湿热蕴结） | 舌质偏红，苔黄腻，脉滑数 |
| 7 | **血瘀质** | 血行不畅，肤色晦暗，唇色偏暗，易有瘀斑，健忘 | 寒邪（寒凝血瘀） | 心、肝（血瘀络阻） | 舌质暗或有瘀点，舌下络脉曲张，脉涩 |
| 8 | **气郁质** | 气机郁滞，情志不畅，忧思多虑，胸胁胀满，善太息 | 风邪（肝风内动） | 肝、心（气机郁滞） | 舌淡红，苔薄白，脉弦 |
| 9 | **特禀质** | 先天禀赋异常，易过敏（鼻炎、哮喘、荨麻疹），对花粉/食物等敏感 | 风邪、燥邪（外邪引触） | 肺、肝（禀赋特异） | 舌象多变，脉象不一 |

### 2.2 体质判定标准

王琦教授体质分类采用**量表评分法**，每个体质类型由若干条目组成，根据条目得分转化为体质得分（0-100分），判定标准如下：

| 得分范围 | 判定结果 |
|----------|----------|
| ≥60分 | 是（判定为该体质类型） |
| 40-59分 | 倾向是（有该体质倾向） |
| <40分 | 否（非该体质类型） |

个体可同时具有多种体质倾向（如气虚质兼阳虚质），以得分最高者为主要体质类型。

### 3.3 矩阵使用说明

1. **确定体质类型**：通过体质问卷（见第4节）获取用户主要体质类型及得分
2. **确定运气格局**：通过 `yunqi-calc` 脚本获取当年大运属性及司天在泉
3. **查矩阵交叉点**：在矩阵中找到对应体质行与运气列的交叉格，获取病机倾向与调理重点
4. **兼顾兼夹体质**：如用户有兼夹体质（如气虚兼阳虚），查两个体质行，综合分析
5. **兼顾不及之年**：如运气推算为"不及"而非"太过"，调理方向反转为"扶其不足"（如木运不及→重点养肝，而非疏肝）
6. **MUST 附加免责声明**：矩阵分析为理论参考，具体调理须由执业中医师辨证论治

## 4. 体质问卷数据接口

### 4.1 JSON Schema 定义

用户体质数据输入采用以下 JSON 格式：

```json
{
  "constitution_type": "阳虚质",
  "constitution_scores": {
    "平和质": 35,
    "气虚质": 50,
    "阳虚质": 78,
    "阴虚质": 25,
    "痰湿质": 30,
    "湿热质": 20,
    "血瘀质": 22,
    "气郁质": 28,
    "特禀质": 15
  },
  "primary_type": "阳虚质",
  "secondary_types": ["气虚质"],
  "questionnaire_version": "WangQi-2009",
  "assessment_date": "2026-06-01",
  "assessed_by": "self-assessment",
  "metadata": {
    "user_age": 35,
    "user_gender": "male",
    "birth_year": 1991,
    "birth_season": "winter",
    "notes": "长期畏寒怕冷，手足不温，喜温饮食"
  }
}
```

### 4.2 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `constitution_type` | string | 是 | 主要体质类型（九种之一） |
| `constitution_scores` | object | 是 | 九种体质得分（0-100），键为体质名，值为得分 |
| `primary_type` | string | 否 | 主要体质类型（与 constitution_type 一致，冗余字段便于程序读取） |
| `secondary_types` | array | 否 | 兼夹体质类型列表（得分40-59的体质） |
| `questionnaire_version` | string | 是 | 问卷版本（标准为 "WangQi-2009"） |
| `assessment_date` | string | 是 | 评估日期（ISO 8601 格式） |
| `assessed_by` | string | 否 | 评估方式（"self-assessment" / "physician" / "auto-derived"） |
| `metadata` | object | 否 | 用户元数据（年龄、性别、出生年份等，用于先天运气分析） |
| `metadata.user_age` | integer | 否 | 用户年龄 |
| `metadata.user_gender` | string | 否 | 用户性别（"male" / "female"） |
| `metadata.birth_year` | integer | 否 | 出生年份（用于推算先天运气格局） |
| `metadata.birth_season` | string | 否 | 出生季节（用于细化先天运气分析） |
| `metadata.notes` | string | 否 | 用户备注（症状描述等） |

### 4.3 得分计算规则

王琦体质量表共60个条目（每个体质类型约6-8个条目），每个条目采用5级评分（1-5分），体质得分转换公式：

```
体质转化分 = (原始分 - 条目数) / (条目数 × 4) × 100
```

- 原始分 = 该体质所有条目得分之和
- 条目数 = 该体质包含的条目数
- 转化分范围 = 0 ~ 100

判定标准：
- 转化分 ≥ 60 → "是"（判定为该体质）
- 转化分 40-59 → "倾向是"（有该体质倾向，列入 secondary_types）
- 转化分 < 40 → "否"（非该体质）

### 4.4 先天运气字段

`metadata.birth_year` 用于推算出生年份的运气格局，作为"先天运气缺陷"分析的输入：

```json
{
  "metadata": {
    "birth_year": 1991,
    "birth_season": "winter"
  }
}
```

- 出生年份（1991，辛未年）→ 大运：水运不及；司天：太阳寒水司天；在泉：太阴湿土在泉
- 先天运气倾向：水运不及则肾气先天偏弱，寒水司天则先天寒气偏盛
- 与后天体质交叉：如后天为阳虚质，则先天寒+后天阳虚，**寒邪根深**，温阳需重用

## 7. 与天气对齐模块的叠加使用

体质对齐与天气对齐可叠加使用，形成"运气 × 天气 × 体质"三维增强分析：

```
运气推算（理论趋势）
  × 天气实况（实际气候）     → 内外邪相合判定
  × 体质数据（个体差异）     → 个体化病机预判
  = 三维增强分析
```

**叠加示例**：
- 运气推算：2026年水运太过，太阳寒水司天（寒邪偏盛）
- 天气实况：上海2月气温3.2℃，低于常年2.8℃，降雪2次 → 对齐"相合"（内外寒邪相合）
- 体质数据：阳虚质（得分78） → 寒邪根深，耐受更差
- **三维结论**：内外寒邪相合 + 阳虚体质 → 寒邪风险最高级别，MUST 重点温阳散寒，建议就医调理

叠加分析时，风险系数计算：

| 叠加维度 | 风险系数 |
|----------|----------|
| 仅运气推算 | ×1.0（基准） |
| 运气 × 天气相合 | ×1.5 |
| 运气 × 体质同向 | ×1.4 |
| 运气 × 天气相合 × 体质同向 | ×2.0（最高级别） |
| 运气 × 天气不符 | ×0.6 |
| 运气 × 体质反向 | ×0.7 |
