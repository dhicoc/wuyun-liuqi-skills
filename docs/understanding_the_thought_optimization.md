# 五运六气技能包 — “帮助人类通过 Agent 理解运气学思想” 优化计划

## 1. 项目定位回顾

本项目的核心价值是：

> 人类将本技能包安装到自己的 AI Agent 中后，Agent 能帮助人类准确、系统、深入地**理解《黄帝内经》运气学这个思想体系**（而非仅仅提供计算结果或临床参考）。

定位关键词：
- **人类**是最终受益者
- **通过 Agent** 进行自然对话
- **理解思想**（宇宙观、时间观、生命观、天人合一等）是核心目标
- 计算、RAG、报告是支撑工具，而非最终目的

当前项目在**准确推算 + 知识检索**上已经很强，但在**帮助人类真正理解运气学思想**这个维度上仍存在明显短板。

---

## 2. 优化目标

通过本计划的实施，让用户在使用本技能时感受到：
- Agent 像一位**耐心的思想讲解者**，而非单纯的计算器
- 能够**渐进式**学习，从概念到整体思想体系
- 对话自然、引导性强、有探索空间
- 安装和启动门槛低，适合非开发者
- 系统会持续优化“教学质量”

---

## 3. 六大优化方向（优先级排序）

### P0（最高优先，立即可见价值）

**1. 学习路径与思想讲解能力**
- 当前偏“给出结果 + 病机参考”
- 目标：增加“思想层”讲解
  - 解释概念背后的哲学（天人合一、气化理论、时间观等）
  - 报告中新增“思想层解读”章节（此格局体现了怎样的宇宙/生命观？）
  - 增加“讲解模式”开关或提示

**4. 安装与启动体验（人类侧）**
- 当前仍是开发者向
- 目标：
  - 新建 `README_for_humans.md`（极简安装 + 如何提问）
  - 在 skill 首次激活时提供 onboarding 流程（Agent 先介绍这个思想 + 本技能能做什么）
  - 简化人类视角的启动引导

### P1（高价值，中期实施）

**2. 更自然的对话入口**
- 支持模糊自然语言输入（“最近运气怎么样”“我这个月要注意什么”）
- 自动路由到合适流程
- 在 README 和 `prompts/onboarding_prompt.md` 中提供丰富的人类提问示例集

**6. 减少“工具感”，增加“思想伙伴感”**
- 输出中增加引导性问题和对话延续
- 年度/个人运气采用叙事化呈现（故事感 + 思想启发）
- 增加“思想地图”或概念关联可视化思路

**3. 渐进式学习体验**
- 报告支持“简化解释版” / “深入版”切换
- “概念连线”功能：解释一个概念时，自动关联经典原文 + 现代比喻 + 相关计算示例

### P2（长期价值）

**5. 反馈与迭代对“理解”的支持**
- 自进化引擎增加“教学质量”反馈维度
- 引导用户反馈：“这个解释帮我理解了吗？”“哪里还觉得模糊？”
- 基于反馈优化解释风格和深度
